# JavaScript GOIT Exercises

Stack: HTML, CSS, JavaScript  
Status: działający  
Demo: https://mikel538.github.io/JavaScript-GOIT-Exercises/

Zbiór wszystkich zadań z kursu Fullstack Developer GOIT w jednej stronie.

Cel projektu:
- utrwalenie wiedzy z JavaScript
- praktyka w manipulacji DOM, funkcjach, eventach i strukturach danych
- organizacja projektów i kodu w jednym repozytorium
- przygotowanie do realnych projektów front-endowych

Funkcjonalności:
- Interaktywna strona zawierająca wszystkie zadania z kursu
- Rozwiązania problemów kursowych z użyciem czystego JavaScript
- Możliwość przeglądania kodu i uruchamiania przykładów w przeglądarce
